# 🐾 Monos Árticos

**Monos Árticos** es un sitio web completo de e-commerce orientado a la venta de productos de invierno y cursos relacionados con deportes de nieve. Incluye una sección de productos dinámicos, un carrito de compras interactivo y un formulario de contacto funcional.

---

## 🌐 Descripción

Este proyecto fue desarrollado con HTML, CSS y JavaScript puro, sin frameworks. El objetivo fue crear un sitio visualmente atractivo, funcional y adaptable, con foco en la experiencia de usuario.

---

## 🚀 Funcionalidades

- 🛒 Carrito de compras dinámico con botones de suma/resta/eliminar por producto.
- 🧤 Catálogo dividido en categorías: ropa, equipamiento y cursos.
- 🎥 Videos embebidos de YouTube para cursos.
- ✉️ Formulario de contacto conectado a [Formspree](https://formspree.io/).
- 💬 Sección de reseñas con imágenes reales y valoraciones.
- 📱 Diseño adaptable (responsive) con CSS Grid y Flexbox.
- 📦 Estructura de productos generados dinámicamente con JavaScript.
- 🎨 Estilos personalizados usando fuentes de Google Fonts y Font Awesome para íconos.
